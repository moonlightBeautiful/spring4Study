    3.AOP
        面向切面编程。主要用于：日志记录、安全管理、事务管理
            连接点：程序运行的运行中，一个一个独立的方法是连接点，连接起来就是程序的运行。
            通知：插入切点后执行的动作。什么和何时。
            切点：连接点的前中后的位置为切点。何处。
            切面：通知和切点的集合
        手动做日志记录：要在每个方法前后手写日志，麻烦。
        AOP做日志：方便
